# Reso — Project Design Context

## Product
Offline-first music player built with Next.js 16 / React 19 / Tailwind v4. Mobile-first layout, single-pane "specimen sheet" aesthetic — every screen feels like a designer's clipboard.

## Phase 1 scope
Upload a single audio file → extract metadata + artwork → play / pause / seek with a transport row that matches the wireframe. The waveform panel is a decorative placeholder for now; the real frequency analyzer ships later.

## Audience
People who care about the *artifact* of a song — sleeve art, tempo, key, the era it came from. Listeners with one song open at a time, full attention.

## Brand personality — "Specimen Sheet"
- Retro-futuristic. Like a 1976 Pioneer hi-fi manual photographed on architect tracing paper.
- Hand-drawn meets mono-typed. Sketchbook + DAW.
- Warm cream paper background with a precise dotted grid — content cards sit *on* the grid, not under it.
- One signature accent: **terracotta / burnt orange** ≈ `oklch(0.62 0.165 35)` (~#C75D2E). Used for state and primary CTAs only.
- Outlined components dominate. Solid fills are reserved for emphasis (RPT active, PSYCH genre, QUEUE primary).

## Type
- **Display (track titles only)**: `Caveat` — variable handwritten serif. Casual, italic-feeling, period-appropriate.
- **Mono (everything else)**: `Space Mono` — labels, time codes, BPM/KEY pills, button text, body copy. Tabular nums always on.
- All chrome copy is **uppercase with letterspacing** (`tracking-[0.18em]+`). Mono caps is the project's voice.

## Color
- Paper: `oklch(0.94 0.018 85)` — warm cream
- Card: same as paper (cards cover the dotted grid where they sit)
- Ink: `oklch(0.22 0.015 50)` — warm dark brown, never black
- Ink soft: `oklch(0.42 0.015 55)`
- Subtle: `oklch(0.62 0.012 65)`
- Accent: `oklch(0.62 0.165 35)` — terracotta
- Accent dim: `oklch(0.50 0.13 35)`
- Border: ink at 0.45 alpha
- Border subtle / dashed: ink at 0.20 alpha

## Layout primitives
- **Stage card**: rounded outer border, "notch" pill at top, square album region (dashed border) above a "WAVEFORM" sub-panel.
- **ControlButton**: square-ish bordered box, ~3.25rem, with an icon or a 4-letter mono caps label (SHFL, RPT). Filled accent variant for "active" state.
- **Pill**: rounded-full outline mono caps. Variants: outline (default), outline-accent (PSYCH), filled-accent (QUEUE), badge (BPM 92 / KEY Am).

## Hard "don'ts"
- No glassmorphism.
- No drop shadows. The aesthetic is matte paper, not glossy.
- No emoji.
- No purple/blue gradients.
- No bouncy easings.
- Don't replace mono caps with sentence-case sans for chrome — that breaks the voice.
